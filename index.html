Meta / Facebook Ads

Meta Ads 3-Tier Architecture & Logic

Meta广告三层架构与投放逻辑
Core Architecture / 核心架构
  • Campaign — Define advertising objective (brand awareness, traffic, conversions, sales) and overall budget strategy. / 定义广告目标（品牌认知、流量、转化、销售等），设定整体预算策略。
  • Ad Set — Define audience (demographics, interests, behaviors, custom audiences, lookalikes), placements, budget allocation, and schedule. / 定义受众（人口统计、兴趣、行为、自定义受众、类似受众）、版位、预算分配、排程。
  • Ad — What users actually see: image/video, copy, CTA button, landing page link. / 用户实际看到的内容：图片/视频、文案、CTA按钮、落地页链接。
Audience Strategy / 受众策略
  • Core Audiences — Manual targeting based on demographics, interests, behaviors. / 基于人口统计、兴趣、行为手动定位。
  • Custom Audiences — Based on website visitors, app events, customer lists, video viewers. / 基于网站访客、APP事件、客户名单、视频观看者。
  • Lookalike Audiences — Find similar people based on custom audiences, 1%-10% similarity. / 基于自定义受众找到相似人群，1%-10%相似度。
  • Advantage+ Audience — Meta AI auto-optimizes targeting (recommended post-2024). / Meta AI自动优化定向（2024年后推荐）。
Budget Strategy / 预算策略
  • CBO — Campaign-level budget, auto-allocated to best-performing Ad Sets. / Campaign层级预算，自动分配给表现最好的Ad Set。
  • ABO — Ad Set-level budget, manual control per audience. / Ad Set层级预算，手动控制每个受众。
  • Test with ABO, scale with CBO. / 测试期用ABO，扩量期用CBO。
Key Metrics / 关键指标
  • CPM — Cost per 1000 impressions, measures traffic cost. / 千次展示成本，衡量流量成本。
  • CPC/CTR — Click cost and rate, measures creative appeal. / 点击成本和点击率，衡量素材吸引力。
  • CPA — Cost per acquisition, measures conversion efficiency. / 单次转化成本，衡量转化效率。
  • ROAS — Return on ad spend, measures profitability. / 广告支出回报率，衡量盈利能力。
  • Frequency — Ad frequency, consider refreshing creative after 3-5 views. / 展示频次，超过3-5次考虑更换素材。
